## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/eniyiweb/web-scraping-portfolio.git
cd web-scraping-portfolio/portfolio

# Install dependencies for any project
cd ecommerce-scraper
pip install -r requirements.txt

# Run the scraper
python main.py --url "https://example.com/products" --output data.csv
```
