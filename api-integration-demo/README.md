# API Integration Demo

Professional API integration examples with REST and GraphQL. Demonstrates authentication, error handling, and data processing.

## 🌟 Features

- ✅ REST API integration with authentication
- ✅ GraphQL queries and mutations
- ✅ API key & OAuth 2.0 handling
- ✅ Rate limiting and retry logic
- ✅ Error handling and logging
- ✅ Webhook receiver example
- ✅ Async/await support

## 📦 Installation

```bash
pip install -r requirements.txt
```

## 🚀 Usage

### REST API Example
```bash
python rest_api_demo.py
```

### GraphQL Example
```bash
python graphql_demo.py
```

### Webhook Server
```bash
python webhook_server.py
```

## 📚 Examples Included

| File | Description |
|------|-------------|
| `rest_api_demo.py` | JSONPlaceholder & GitHub API integration |
| `graphql_demo.py` | GitHub GraphQL API queries |
| `webhook_server.py` | Flask webhook receiver |
| `async_api_client.py` | Async HTTP client with aiohttp |

## 🔐 Authentication Methods

- API Key (Header & Query param)
- Bearer Token
- OAuth 2.0
- Basic Auth

## 📊 Use Cases

- Third-party API integration
- Data synchronization
- Real-time webhook processing
